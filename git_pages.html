使用VScode當編輯器

 
另一說法(來自youtube 紅色死神的影片- search key : github pages 教學) => 此方法方便在VS code 中編輯後push to github)
1. create an account on GitHub 
2. Get a URL after your complete account created.  copy the URL . 

進到Visual code , 安裝git extention 套件 , "GitHub pull requests and issues" 
裝完後,  Open Git tool on the left side , 點選" Clone Repository" , 然後將之前copy 之URL 貼網址 
select repository allocation 要放在本機的拿一個資料夾
Open in new windows in VScode .
Create index.html in vscode
switch to Git , click "commit all" in command list . 
keyin some message in message bar  
click "push to ..." in command list and choose provious URL．（push completed)
switch to github . copy 專案名稱並貼到網址上即可看到所創建的網頁

先clone再push

1. create an account on Github website 
2. build a "Repository" 
login github -> new repository -> 填入repository name , example : 
[你的Github帳號].github.io -> click "create repository" botton to create 

3. 就可以看見你所創建的儲存庫名稱了

and than you can do , 

1. Clone repository from Github via https portocal in terminal . (下指令來達成)
steps:
git clone http://github.com/kevinattao/kevinattao.github.io.git   (由github 下載至本機)

2. push project to github 
steps :
1. 進入下載下來的目錄中 and create a index.html file 
2.git add . 
3.git commit -m 'first commit'
4.git push 

Now, you have alreadly ceated your HOME page. and then you can create other project via new another folder with the name of "other-project"
and push to your repo.
